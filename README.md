# qpros-task


A Selenium test script and Express API developed as part of an interview task for QPros.


## Installation

Clone the repository: git clone https://github.com/ksuhiyp/qpros-task.git
Navigate to the project directory: cd qpros-task
Install dependencies:

```bash
npm install
```

## Build

```bash
npm run build
```

## Run Server

```bash
npm run start
```
